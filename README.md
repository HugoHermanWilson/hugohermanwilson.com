# hugohermanwilson.com
Redoing hugohermanwilson.com using Netlify and GatsbyJS

Made using Node v10.15.1, npm 6.14.5

# Getting Started:

1) Run `npm install`
3) Run `gatsby develop` to launch the project on port 8000
